# unitree_ros2 Digest

Source: https://github.com/unitreerobotics/unitree_ros2

## Captured Facts

- Official ROS2 communication package for Unitree robots.
- README states tested environment: Ubuntu 20.04 with ROS2 Foxy.
- Communication depends on DDS transport and network interface configuration.
- G1 examples include joint-state reading and high-level command executables.

## Use in This Repo

- Bridge reference for ROS2-based telemetry/control workflows.
- Useful when integrating policy outputs into ROS2 nodes or mixed stacks.
